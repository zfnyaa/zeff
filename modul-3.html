HTML & CSS

1. HTML (HyperText Markup Language):

HTML adalah bahasa markup yang digunakan untuk membuat struktur dan konten dari sebuah halaman web. Dengan HTML, kita dapat membuat berbagai elemen seperti teks, gambar, tautan, tabel, formulir, dan lainnya.

  • HTML terdiri dari elemen-elemen yang diatur dalam tag-tag seperti <h1>, <p>, <img>, <a>, dll.
  • HTML bertanggung jawab atas struktur dan konten sebuah halaman, tetapi tidak mengatur tampilan atau desainnya.

2. CSS (Cascading Style Sheets):

CSS adalah bahasa yang digunakan untuk mendesain dan mengatur tampilan elemen-elemen yang dibuat dengan HTML. CSS memungkinkan kita untuk mengatur warna, font, layout, dan tata letak halaman web agar terlihat lebih menarik.

  • CSS dapat diterapkan secara langsung di dalam tag HTML (inline), di dalam tag <style> (internal), atau di file terpisah dengan ekstensi .css (eksternal).
  • Dengan CSS, kita bisa mengubah tampilan seluruh halaman web secara konsisten dan fleksibel, tanpa harus menulis ulang banyak kode.
