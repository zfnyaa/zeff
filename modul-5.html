Penggunaan JavaScript untuk Menampilkan Data

1. Menggunakan console.log()

Ini adalah cara yang paling sering digunakan oleh developer untuk menampilkan data di konsol browser. Fungsinya lebih untuk debugging daripada menampilkan data langsung ke pengguna. Untuk melihat hasilnya, Anda bisa membuka Konsol di alat pengembang browser (biasanya dengan menekan F12 atau Ctrl + Shift + I di browser).

2. Menggunakan prompt()

prompt() akan menampilkan jendela dialog kepada pengguna yang memungkinkan mereka untuk memasukkan data, lalu mengembalikan input tersebut.

3. Menggunakan console.table()

Jika Anda ingin menampilkan data dalam bentuk tabel (misalnya array atau objek), Anda bisa menggunakan console.table() untuk memvisualisasikan data dalam format tabel di konsol.
